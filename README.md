# 💼 Elite Developer Portfolio – Data-Driven Architecture

My professional digital home and personal portfolio website. This project is built to showcase my technical skills, completed projects, and frontend expertise to recruiters and collaborators with a clean, premium aesthetic.

## ✨ Key Features

* **Data-Driven Architecture:** Project details, skills, and bio are separated and fetched dynamically using a `profile.json` dataset via the JavaScript Fetch API rather than being hardcoded into the HTML.
* **Premium Dark Theme:** A minimalist, eye-friendly layout featuring custom-styled layouts, smooth interactions, and fluid typography.
* **Highly Responsive Design:** Optimized meticulously for all viewport sizes (Mobile, Tablet, and Desktop) ensuring a top-tier user experience.
* **Scalable Components:** Built cleanly so that adding a new project or updating a skill is as easy as adding a single line to a JSON file!

## 🛠️ Tech Stack

* **Frontend:** Semantic HTML5, CSS3 (Custom Flexbox & Grid layouts)
* **Logic & Rendering:** Vanilla JavaScript (ES6+), Fetch API
* **Data Source:** JSON (JavaScript Object Notation)
